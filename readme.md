# Biscuits Industrial
Website project for Eve Online.

Starting the project on your local machine.
``npm server.jar``

Modules used so far


[Nodemon](https://www.npmjs.com/package/nodemon)
``npm i nodemon --save``

[Express](https://www.npmjs.com/package/express)
``npm i express --save``

[pugjs](https://www.npmjs.com/package/pug)
``npm i pug --save``
