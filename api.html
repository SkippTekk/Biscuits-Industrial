Hello! You have found the API page. Here are your instructions to see everything this API does.

To find things in the InvTypes database. the following URl is provided as;

/api/InvTypes/s/NUMBER

for example

/api/InvTypes/i/2161 brings us to the Crucifier. Notice the i in the URL? That means ID

Now to get the Ship directly. It's very case sensitive!

/api/InvTypes/s/Crucifier brings us the Crucifier BUT with the ship name. Basically for new people that don't know the ID. Notice the s in the URL? That means SHIP.
